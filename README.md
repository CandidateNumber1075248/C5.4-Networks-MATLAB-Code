**C5.4 Networks Mini-Project**

This repository contains all MATLAB code developed for the C5.4 Networks Mini-Project.

All scripts must be in the same working directory for the figure scripts to locate `aon_network.m`.

Each figure script is self-contained and will produce the corresponding figure as it appears in the mini-project report, correct up to random variation inherent to the all-or-none network.

`aon_network.m`	Function that generates an all-or-none network for a specified number of nodes and copying probability .

`Figure1.m`	Generates Figure 1 from the mini-project report.

`Figure2.m`	Generates Figure 2 from the mini-project report.

`Figure3.m`	Generates Figure 3 from the mini-project report.

`Figure4.m`	Generates Figure 4 from the mini-project report.

`Figure5.m`	Generates Figure 5 from the mini-project report.

`Figure6.m`	Generates Figure 6 from the mini-project report.

`Figure7.m`	Generates Figure 7 from the mini-project report.
